# Tests

Tests stay close to the code they verify:

- `apps/api/tests`: Pytest unit, graph, API, RBAC, tenant, webhook, report, and AI Operations coverage.
- `apps/web/tests/e2e`: Playwright browser vertical slice and mobile coverage.
- `apps/web/tests/portfolio`: Reproducible synthetic screenshot capture.

See [docs/testing-report.md](../docs/testing-report.md).
