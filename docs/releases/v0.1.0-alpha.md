# OrbitOps v0.1.0-alpha

The first public alpha demonstrates a complete approval-first business automation vertical slice.

## Highlights

- **Authentication and RBAC:** JWT sessions, tenant scope, Admin/Manager/Agent Viewer permissions.
- **LangGraph workflows:** deterministic sales scoring, multi-model research and drafting, human approval, report generation.
- **Communication delivery layer:** email/WhatsApp lifecycle events, signed webhooks, duplicate/replay protection, retries, dead letter, and reply classification.
- **AI Operations:** reliability, latency, retries, tokens, cost, evaluations, feedback, prompts, routing, and playground comparisons.
- **Governance:** immutable audit events, approval idempotency, and provider-event history.
- **Quality:** backend tests, production frontend build, Playwright vertical slice, security and CI workflows.

## Safe defaults

Live LLM calls and real message delivery are disabled. Configure provider credentials through environment/secret management and complete sandbox certification before enablement.

## Known alpha boundaries

- Automated API tests use SQLite; production PostgreSQL integration remains a release gate.
- Redis/SQS resilience and load/security testing require a production-like environment.
- Refresh-token revocation and enterprise SSO are not included.
- Provider credentials and production adapters are not bundled.

See the [testing report](../testing-report.md) and [deployment guide](../deployment.md).
